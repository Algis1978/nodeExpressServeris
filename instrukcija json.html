JSON instrukcija su serveriu

  • json yra teksto byla, kurioje aprašyta objektų struktūra.
  • Iš principo daro, ką darė express-handleriai, tas pats kitaip.
  • Sukūrus naują endpointą
    • app.get ("/json/zmones", (req, res) => {
      //Nurodo naršyklei atsiunčiamo teksto formatą (json)
      res.set("Content-Type", "application/json")
      //JSON.stringify sukelia duomenis į JSON formatą.
      res.send(JSON.stringify(zmones))
      });
    • Į naršyklės užklausą "/json/zmones" bus nusiųsti objekto "zmones" duomenys.
  • fetch funkcija parsisiunčia duomenis neperkranant html.
  • JS byloje (index.js) susikuriama funkcija (getZmones), kuri sufetchins duomenis iš "json/zmones"
  • HTML byloje sukuriamas mygtukas, kuris onclick iškvies šią funkciją.
  • Susikuriama funkcija, kuri išves duomenis į naršyklės langą (showZmones). Kartu sukuriami atskiri trnimo mygtukai.
  • Po to susikuriama funkcija, kuri neleis išvesti duomenų daugiau nei vieną kartą paspaudus mygtuką (cleanNode)
  • Po to sukuriama funkcija, kuri tvarkys delete mygtukus (deleteZmogus) per id identifikuotojus.
  • run.js byloje pridedamas app.delete("/json/zmones:id?,) metodas.
  • Susikuriama funkcija įrašų sukūrimui (addZmogus)
  • Pridedamas app.post metodas į run.js, neaišku, ko.
